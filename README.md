# BackEndPractice
Tis project is build using Nodejs and express framework.

## Packages Used

* "dependencies"
* "axios",
* "bcrypt"
* "cors"
* "express"
* "jsonwebtoken"
* "mongoose"
* "nodemailer"
 
## Run project 
* npm install 
* npm start

## Features
* Authentication using JWT
* Role based authorization 
* Implementaion of nodemailer
